Query para detectar si un paciente esta sacando turnos seguidos.
---------------------------------
select tr.fechaTurno,p.dni,p.nombre,p.apellido,tr.paciente, count(*) as 'C'
from turno_registrados tr join pacientes p on tr.paciente=p.id 
where tr.fechaTurno>'2020-01-06' and tr.fechaTurno<'2020-03-01' and tr.activo = 1 and tr.primerControl='NO'
group by tr.paciente
order by C desc
---------------------------------
Query para detectar si un turno fue sacado mas de una vez por algun timeout.    
---------------------------------
SELECT count(paciente),fechaTurno, paciente, horario, primerControl FROM turno_registrados where paciente!=84 and activo=1 and primerControl='NO'
group by fechaTurno,paciente,horario  
ORDER BY `count(paciente)`  ASC
